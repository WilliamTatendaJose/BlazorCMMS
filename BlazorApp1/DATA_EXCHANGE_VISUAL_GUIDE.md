## ?? Design System

### Typography
- **Page Title**: Large, Bold, Primary Color + Icon
- **Section Headers**: Medium, Bold, Accent Color + Icon
- **Buttons**: Regular, Medium Weight, Colored Background + Icon
- **Description**: Small, Light Gray, Regular Weight

### Spacing
- **Sections**: 24px gap between major sections
- **Cards**: 20px padding inside
- **Buttons**: 12px padding
- **Icons**: 4px spacing from text

### Colors
- **Primary**: #0288d1 (Blue)
- **Success**: #4caf50 (Green)
- **Warning**: #ff9800 (Orange)
- **Danger**: #e53935 (Red)
- **Info**: #29b6f6 (Cyan)

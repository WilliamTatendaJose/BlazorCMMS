## Common Issues

### ? Upload button disabled?
- Select a file
- Enter a title
- Select a category
- Then it will enable

### ? File too large?
- Max file size is 50MB
- Choose a different file
- Or compress existing file

### ? File type not supported?
- Use PDF, Word, Excel, Images, or CAD files
- Other formats not accepted

### ? Upload failed?
- Check internet connection
- Verify file size
- Try again
- Contact admin if persists

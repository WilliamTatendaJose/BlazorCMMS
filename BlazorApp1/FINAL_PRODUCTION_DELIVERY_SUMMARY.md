## ?? How to Deploy

### Option 1: Docker
```bash
docker build -t rbm-cmms .
docker run -p 8080:80 rbm-cmms
```

### Option 2: IIS (Windows)
1. Publish: `dotnet publish -c Release -o ./publish`
2. Copy to IIS folder
3. Create IIS website pointing to publish folder
4. Configure application pool

### Option 3: Linux
1. Install .NET 10 runtime
2. Publish: `dotnet publish -c Release -o ./publish`
3. Copy to server
4. Configure systemd service
5. Configure Nginx reverse proxy

### Option 4: Azure
1. Create App Service (.NET 10)
2. Create SQL Database
3. Deploy from GitHub using GitHub Actions
4. Configure connection strings
5. Run migrations

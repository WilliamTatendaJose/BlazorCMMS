## ?? Usage Tips

### Exporting Data
1. Click any export button
2. File downloads automatically
3. Format: `{DataType}_{YYYYMMDD_HHMMSS}.{ext}`
4. Supported: CSV, Excel (.xlsx), JSON

### Importing Data
1. Go to "Data Import" section
2. Click "Choose CSV"
3. Select your file
4. Click "Import"
5. Check success message

### Using Templates
1. Go to "Import Templates"
2. Click "Download Template"
3. Open in Excel
4. Fill in your data
5. Save as CSV
6. Import using Data Import

## ?? File Formats

### CSV
- **Extension**: .csv
- **Format**: Comma-separated values
- **Use**: Import/Export, universal
- **Size**: Small (~50KB per 100 records)

### Excel
- **Extension**: .xlsx
- **Format**: Modern Excel
- **Use**: Professional reports
- **Features**: Formatting, colors, merged cells
- **Size**: Medium (~100KB per 100 records)

### JSON
- **Extension**: .json
- **Format**: Structured text
- **Use**: APIs, integration
- **Size**: Large (~200KB per 100 records)

## Tips & Tricks

### ?? Tip 1: International Reporting
When sending reports internationally, use **Metric** (most common globally)

### ?? Tip 2: API Integration
If integrating with external systems, note that data is stored in **Imperial**

### ?? Tip 3: Team Coordination
If your team uses different units, take screenshots showing your unit system

### ?? Tip 4: Historical Consistency
The same reading viewed in different units will show mathematically equivalent values

### ?? Tip 5: Precision
All values displayed with 1 decimal place for readability

## Troubleshooting Guide

### Issue: Units selector not showing
```
? Problem: Can't find Units selector

? Solution:
   1. Check top-right of page header
   2. Look near "Add Reading" button
   3. Refresh page if still not visible
   4. Check browser console for errors
```

### Issue: Values not converting
```
? Problem: Temperature shows 85.0°F even after selecting Metric

? Solution:
   1. Refresh the page
   2. Re-select the unit system
   3. Check that you're viewing a saved reading
   4. Try in a different browser
```

### Issue: Wrong conversion value
```
? Problem: 85°F shows as 30°C (should be 29.4°C)

? Solution:
   1. This is likely rounding during display
   2. Actual calculation is: (85-32)×5/9 = 29.4°C
   3. Values show 1 decimal place
   4. This is expected behavior
```

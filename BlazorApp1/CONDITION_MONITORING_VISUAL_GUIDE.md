## ?? Color Coding System

### Health Score Colors
```
GREEN (#43a047)     80-100%  ? Healthy
ORANGE (#fb8c00)    60-79%   ?? Warning
RED (#e53935)       <60%     ?? Critical
```

### Parameter Status
```
Temperature:
?? GREEN: 40-130°F (Normal)
?? ORANGE: >130 or <40 (Warning)
?? RED: Extreme values (Critical)

Vibration:
?? GREEN: 0-5 mm/s (Normal)
?? ORANGE: 5-10 mm/s (Warning)
?? RED: >10 mm/s (Critical)

Pressure:
?? GREEN: 20-100 PSI (Normal)
?? ORANGE: >100 or <20 (Warning)
?? RED: Extreme values (Critical)
```

### Reading Status
```
?? Normal: Green left border, green badge
?? Warning: Orange left border, orange badge
?? Critical: Red left border, red badge
```

## ?? Data Flow Diagram

```
User Interface
    ?
    ?? Asset Selection
    ?  ?? Triggers: OnAssetSelected()
    ?     ?? Loads asset details
    ?     ?? Loads condition readings
    ?
    ?? Record Parameters
    ?  ?? Real-time validation
    ?     ?? Temperature check
    ?     ?? Vibration check
    ?     ?? Pressure check
    ?
    ?? Save Reading
       ?? SaveReading()
          ?? Determine status
          ?? Record user attribution
          ?? Save to database
          ?? Reload readings
          ?? Update metrics
          
Dashboard Metrics
?? Total Readings (sum of all)
?? Active Alerts (count)
?? Monitored Assets (count)
?? Critical Status (count)
?? Today's Readings (filtered by date)

Data Display
?? Health Score ? Color coded
?? Recent Readings ? Sorted by date
?? Trends ? Calculated averages
?? Alerts ? Generated from readings
```

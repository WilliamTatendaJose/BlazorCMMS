## ?? Asset Selection Workflow

1. **Select Asset** ? Dropdown at top
2. **Auto-Load:**
   - Asset details
   - Health score
   - All readings (ordered by date DESC)
   - Generate alerts
   - Update all sections
3. **Ready to Record:** Form ready for new reading

## ?? Recording a Reading

### Step 1: Select Asset
```html
<select @bind="selectedAssetId">
  -- Select Asset --
  PMP-001 - Hydraulic Pump
  PMP-002 - Secondary Pump
</select>
```

### Step 2: Fill Form
```
Temperature:  85.5 °F     ? Normal
Vibration:     3.2 mm/s   ? Normal
Pressure:     50.0 PSI    ? Normal
Oil Analysis: Normal      ?
```

### Step 3: Save
- Button becomes enabled when asset selected
- Click "?? Save Reading"
- Shows "? Saving..."
- Auto-resets form
- Shows success message
- Updates all sections

## ??? Parameter Thresholds

### Temperature (°F)
- ?? Critical: < 40 or > 130
- ?? Warning: -
- ?? Normal: 40-130

### Vibration (mm/s)
- ?? Critical: > 10
- ?? Warning: 5-10
- ?? Normal: 0-5

### Pressure (PSI)
- ?? Critical: < 20 or > 100
- ?? Warning: -
- ?? Normal: 20-100

### Oil Analysis
- ?? Critical: Critical
- ?? Warning: Warning
- ?? Normal: Normal

## ?? Alert Types

### Health Score Alert
```
Triggers: HealthScore < 60%
Severity: CRITICAL
Message: "Asset health is at XX% - Consider maintenance"
```

### Overdue Maintenance
```
Triggers: NextScheduledMaintenance < Today
Severity: CRITICAL
Message: "Maintenance was due on MMM dd, yyyy"
```

### Abnormal Parameters
```
Triggers: Latest reading status = Warning
Severity: WARNING
Message: "Latest reading shows warning-level parameters"
```

### Critical Parameters
```
Triggers: Latest reading status = Critical
Severity: CRITICAL
Message: "Latest reading shows critical-level parameters"
```

## ?? Interactive Features

### Real-Time Validation
As you type in the form:
- Temperature field auto-validates
- Shows color-coded status
- Updates message instantly
- Same for Vibration, Pressure, Oil Analysis

### Click to Details
- Click any reading card ? Modal opens
- Shows all parameters
- Shows notes
- Click Close ? Modal closes

### Hover Effects
- Reading cards scale up slightly
- Buttons change shade
- Badges highlight

## ? Performance Tips

1. **Asset Selection:** Choose asset first before recording
2. **Export Data:** Export regularly to keep readings backed up
3. **Clear Selection:** Clear unused asset selection
4. **Refresh:** Use Refresh if data seems stale

## ??? Troubleshooting

| Issue | Solution |
|-------|----------|
| Form not enabling | Select an asset first |
| Readings not showing | Select asset, check readings exist |
| Alerts not appearing | Check asset health < 60% or maintenance overdue |
| Modal won't close | Click the X button or outside modal |
| Export not working | Check readings exist first |

## ?? Transitions

All color changes use smooth CSS transitions:

```css
transition: background-color 0.3s ease, color 0.3s ease;
```

### What Happens When You Toggle

1. **0ms**: User clicks theme toggle
2. **0-300ms**: Colors smoothly fade to new values
3. **300ms**: All elements now use new theme colors
4. **Instant**: Preference saved to browser

## ?? Best Practices

### Light Mode is Best For:
- ? Daytime use
- ? Bright environments
- ? Printed documents
- ? Long reading sessions

### Dark Mode is Best For:
- ? Night time use
- ? Reduced eye strain
- ? Low light environments
- ? Extended screen time

## ?? CSS Variables Used

All colors follow this pattern:

```css
/* Light mode (default) */
:root {
    --rbm-primary: #37474f;
    --rbm-accent: #0288d1;
    --rbm-bg: #eceff1;
    --rbm-card-bg: #ffffff;
    --rbm-text: #263238;
    --rbm-text-light: #607d8b;
    --rbm-border: #cfd8dc;
}

/* Dark mode (when html.dark-mode is active) */
html.dark-mode {
    --rbm-primary: #1a1a1a;
    --rbm-accent: #42a5f5;
    --rbm-bg: #121212;
    --rbm-card-bg: #1e1e1e;
    --rbm-text: #e0e0e0;
    --rbm-text-light: #b0b0b0;
    --rbm-border: #333333;
}
```

## ?? Color Contrast

Both themes maintain WCAG AA accessibility standards:

**Light Mode**
- Text on white: 10.8:1 contrast ratio ?
- Text on light grey: 8.2:1 contrast ratio ?

**Dark Mode**
- Text on black: 11.5:1 contrast ratio ?
- Text on dark grey: 7.9:1 contrast ratio ?
